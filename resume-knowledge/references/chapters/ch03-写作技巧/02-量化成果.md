# ch03-02 量化成果

> 数字是 HR 判断能力的唯一客观依据。能量化的全部量化。

---

## 各岗位量化方向

| 岗位 | 可量化指标 |
|------|-----------|
| **行政/助理** | 整理文件 X 份、归档准确率、节省时间 |
| **人力/招聘** | 筛选简历 X 份、到岗 X 人、招聘周期缩短 X 天 |
| **新媒体/运营** | 发布推文 X 篇、阅读量 X、粉丝增长 X% |
| **技术** | 发现漏洞 X 个、修复率、避免损失 X 万 |
| **销售** | 成交 X 单、GMV X 万、转化率从 X% 到 X% |
| **产品** | 上线 X 个功能、DAU 增长 X%、留存率提升 X% |

---

## 量化的三个层次

| 层次 | 示例 | 效力 |
|------|------|------|
| **模糊** | "大大提升了效率" | ❌ 无效 |
| **单独数字** | "效率提升 80%" | ⚠️ 弱 |
| **对比数字** | "从 30 分钟缩短至 6 分钟" | ✅ 强 |

**对比数字 > 单独数字 >>> 模糊描述**

---

## 没有"大数字"怎么办？

- 金额小 → "在 5 万元预算内完成 XX"
- 时间短 → "3 天内交付 XX"
- 规模小 → "从 0 到 1 搭建 XX 体系"
- 人数少 → "带领 3 人团队完成 XX"
