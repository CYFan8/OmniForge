# ch05-02 终极自检清单

> 每次投递前，跑一遍这个清单。5 分钟，少犯 80% 的错误。

---

## 格式检查

```
□ 全部在一页内
□ 保存为 PDF（命名：姓名-岗位-学校-电话.pdf）
□ 全篇不超过 2 种字体
□ 全篇不超过 3 种颜色
□ 没有使用文本框/表格/图标
□ 页边距 ≥ 2cm
□ 时间格式统一（2024.06 - 2024.09 或 2024.6-2024.9）
```

## 内容检查

```
□ 每条经历有动词+数据
□ 没有"参与""协助"等弱动词
□ 没有"我""的""了"等冗余词
□ 没有自我评价废话
□ 没有与岗位无关的信息（政治面貌/籍贯/星座）
□ 联系方式正确（电话+邮箱）
□ LinkedIn/作品链接有效
```

## ATS 检查

```
□ 记事本粘贴测试通过
□ JD 核心关键词在正文区
□ 没有用侧边栏/文本框放重要信息
□ 缩写首次出现有全称
```

## 最终检查

```
□ 找一个人帮你看一遍（fresh eyes 总能发现你漏掉的）
□ 大声读一遍（耳朵能听出眼睛忽略的错误）
```
