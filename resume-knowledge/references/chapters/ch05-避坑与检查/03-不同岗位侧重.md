# ch05-03 不同岗位的简历侧重

> 不同岗位，HR 看的东西完全不一样。

---

## 各岗位重点

| 岗位类型 | 最看重 | 简历重点 |
|---------|--------|---------|
| **技术岗** | 技术栈+项目 | 项目经历 > 技能列表 > 实习 |
| **产品岗** | 产品思维+数据 | 项目中的用户洞察、数据决策 |
| **运营岗** | 数据增长+执行力 | 具体的增长数字、活动 ROI |
| **市场岗** | Campaign+ROI | 操盘过的项目、预算、效果 |
| **设计岗** | 作品集+审美 | 作品链接放最前面 |
| **金融岗** | 证书+实习 | CPA/CFA+券商/银行实习 |
| **咨询岗** | 逻辑+分析 | 案例经历、分析框架 |
| **管培生** | 综合潜力 | 领导力+快速学习+多元经历 |

---

## 简历板块排序调整

```
技术岗：技能 → 项目 → 实习 → 教育
产品/运营：实习 → 项目 → 技能 → 教育
设计岗：作品链接 → 经历 → 技能 → 教育
应届生：教育 → 实习 → 项目 → 技能
```
