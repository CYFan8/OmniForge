# ch04-02 一岗一稿策略

> 拒绝海投。每份简历针对性修改 = 面试率翻倍。

---

## 主简历 + 定制版

```
主简历（完整版）: 包含所有经历，3-5页，只给自己看
         ↓
定制版（投递版）: 从主简历中挑选最相关的经历，1页
```

---

## 针对不同岗位的调整

| 投递岗位 | 调整内容 |
|---------|---------|
| 产品经理 | 突出用户研究、数据分析、需求文档 |
| 运营 | 突出活动策划、数据增长、内容输出 |
| 技术 | 突出项目架构、性能优化、技术栈 |
| 市场 | 突出 campaign 策划、ROI 数据、渠道管理 |

---

## 定制检查

每次投递前确认：
```
□ JD 前 5 个核心关键词都出现在简历中了？
□ 最相关的经历排在该模块第一条？
□ 技能区列的技能与岗位匹配？
□ 自我定位语针对这个岗位改了吗？
```
