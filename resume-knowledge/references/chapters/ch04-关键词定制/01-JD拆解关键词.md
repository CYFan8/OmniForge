# ch04-01 从 JD 拆解关键词

> JD = 岗位说明书。它免费告诉了你 HR 想要什么。

---

## 三步拆解法

### Step 1: 高亮所有名词（硬技能）
```
Python、SQL、Figma、数据分析、项目管理、用户研究
→ 这些必须出现在你的技能区或经历中
```

### Step 2: 高亮所有动词（软技能/方法论）
```
策划、统筹、跨部门协作、从0到1、A/B测试
→ 这些必须嵌入经历描述的 Action 部分
```

### Step 3: 高亮"加分项"
```
"有XX经验者优先""熟悉XX者加分"
→ 如果你有，一定写上；没有，评估是否值得补
```

---

## 匹配度计算

```
你的匹配词数 / JD 关键词总数 = 匹配率
```

目标：**80-85%**。太高=像抄的，太低=不匹配。
