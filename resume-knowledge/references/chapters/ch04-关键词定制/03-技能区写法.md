# ch04-03 技能区的写法

> 技能区不是"我会什么"的购物清单，而是"我能为岗位解决什么问题"的证据清单。

---

## 技能分类法

| 类别 | 例子 | 写法 |
|------|------|------|
| **工具** | Python、Figma、Excel | 直接列出 |
| **领域知识** | 用户增长、SEO | 与经历呼应 |
| **语言** | 英语 CET-6 (580) | 带分数/证书 |
| **证书** | CFA L2、PMP | 简称+时间 |

---

## 避免

```
❌ 精通 Office → 人人都会
❌ 吃苦耐劳、性格开朗 → 没人信
❌ 熟练掌握 PS、AI、PR、AE、Figma、XD... → 像在报菜名
```

## 推荐

```
✅ 熟练运用 Excel 数据透视表与 VLOOKUP 完成销售分析
✅ Python（Pandas/Matplotlib）：独立完成用户行为数据清洗与可视化
✅ 英语 CET-6 (588)，可作为工作语言
```

---

## 技能数量

**列出与岗位最匹配的 5-8 项**。太多 = 不精，太少 = 能力不足。
