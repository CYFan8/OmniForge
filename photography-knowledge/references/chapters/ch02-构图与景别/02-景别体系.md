# ch02-02 景别体系

> 越紧张越靠近，越舒缓越拉开。景别是情绪的晴雨表。

**口诀**：远景建世界，全景看关系，中景推叙事，近景捕情绪，特写揭本质，大特写炸裂

---

## 六级景别

| 景别 | 英文 | 取景 | 功能 | 使用 |
|------|------|------|------|------|
| **大远景** | Extreme Long Shot | 人物极小，环境宏大 | 建世界观，孤寂/宏大 | 开场、史诗结尾 |
| **全景** | Long Shot | 全身+场景 | 空间关系，人物互动 | 群戏、动作 |
| **中景** | Medium Shot | 上半身+背景 | **叙事主力** | 对话、日常 |
| **近景** | Close-up | 头部+肩部 | 微表情，共情 | 情感戏 |
| **特写** | Extreme CU | 面部/局部 | 情感冲击 | 关键揭示 |
| **大特写** | Macro | 极致放大（眼/嘴） | 视听冲击 | 惊悚、高潮 |

---

## 情绪递进曲线（以《十二怒汉》为例）

```
开场 → 中远景：开阔，轻松
争论 → 中近景/特写连续出现：压迫递增
高潮 → 大特写：冲突顶峰
缓和 → 回到中景：空间舒展，解脱
```

**规律**：景别是情绪的刻度尺。不需要台词，景别变化本身就告诉观众"紧张起来了"。

---

## 特殊景别

| 类型 | 效果 |
|------|------|
| **过肩镜头 (OTS)** | 观众=对话参与者 |
| **鸟瞰镜头** | 上帝视角、全局审视 |
| **POV 主观** | 观众=角色，最强共情 |
| **荷兰角 (Dutch Angle)** | 倾斜水平线=不安/混乱 |
