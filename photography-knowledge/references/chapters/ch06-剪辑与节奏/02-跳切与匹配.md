# ch06-02 跳切与匹配

> 打破规则的前提：你知道规则是什么。

---

## 跳切（Jump Cut）

**操作**：同一主体、同一角度，时间跳跃。

### 效果
- 时间流逝感
- 焦虑、不安、破碎
- 现代 Vlog 的口语化剪辑语言

### 何时用
- 表现角色心理断裂
- 压缩无聊的等待过程
- 故意制造不舒适感

### 何时不用
- 正常对话场景 → 会让观众困惑
- 追求"好莱坞流畅感"时

**起源**：戈达尔《筋疲力尽》— 开创性使用，改变了电影语法。

---

## 匹配剪辑（Match Cut）

画面元素在前后镜头中对应——最震撼的转场方式。

### 形状匹配
```
圆形物体 A → 圆形物体 B
（如《2001》骨头→飞船）
```

### 运动匹配
```
人物向上跳 → 切 → 火箭升空
```

### 声音匹配
```
前一个镜头的声音 → 延续到下一个镜头的画面
```
