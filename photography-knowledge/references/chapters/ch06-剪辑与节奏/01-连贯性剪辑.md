# ch06-01 连贯性剪辑

> 好莱坞标准：让观众"感觉不到剪辑的存在"。

**口诀**：动接动，静接静，情绪接情绪

---

## 五大法则

| 法则 | 要点 |
|------|------|
| **动接动** | 上一个镜头有运动 → 下一个继续运动 |
| **静接静** | 上一个镜头静止 → 下一个也静止 |
| **视线匹配** | A 看画右 → 切 B 在画左，视线相对 |
| **30 度法则** | 相邻镜头角度差 ≥ 30°，否则画面"跳" |
| **动作剪辑点** | 在动作进行中切，不在开始/结束时切 |

---

## 剪辑点的选择

```
错误：————[动作完成]|切|————
正确：————[动作中途]|切|[继续]————
```

观众的眼睛在动作进行中被引导，注意不到剪辑。

---

## 节奏控制

| 镜头时长 | 节奏 | 场景 |
|---------|------|------|
| < 1 秒 | 极快、冲击 | 动作高潮 |
| 1-2 秒 | 紧张、能量 | 追逐、打斗 |
| 3-5 秒 | 正常叙事 | 对话 |
| 5-10 秒 | 舒缓 | 情感酝酿 |
| > 10 秒 | 沉浸 | 长镜头 |
