# ch06-04 剪辑的叙事节奏

> 一部电影的剪辑节奏，就是它的心跳。

---

## 经典三幕节奏

```
第一幕（建置）25%      第二幕（对抗）50%      第三幕（解决）25%
    慢→中速                中速→加速               快→极慢（余韵）
  引入、铺垫              冲突升级、转折           高潮、结局
```

---

## 章节式节奏渐变（《杀手》案例）

芬奇用剪辑节奏外化杀手心理状态：

```
第一章：绝对平稳，井然有序
第二章：植入节奏变量
第四章：彻底质变——跳接、加速、冗余镜头消失
终结篇：稳定节奏回归 = 杀手重拾控制
```

**形式 = 内容**。剪辑节奏本身就是叙事。

---

## 实战节奏清单

```
□ 开场 3 分钟是否建立了核心悬念？
□ 每个场景的"进点"和"出点"是否有效？
□ 有没有可以再删 1 秒的镜头？（答案几乎总是"有"）
□ 节奏是否在变化？（一成不变 = 催眠）
□ 最长镜头和最短镜头之间的对比够大吗？
□ 高潮段落的镜头时长是否明显短于其他段落？
```
