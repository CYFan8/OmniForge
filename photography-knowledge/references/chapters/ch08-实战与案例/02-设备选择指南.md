# ch08-02 设备选择指南

> 最好的设备是你手头那台。但知道该升级什么可以省很多钱。

---

## 稳慢匀 — 三字基本功

| 基本功 | 要点 |
|--------|------|
| **稳** | 降低重心、手肘贴身体、小碎步、膝盖缓冲 |
| **慢** | 新手宁可慢 20%，不要快 1% |
| **匀** | 起幅落幅完整、速度无突变、呼吸平稳 |

---

## 设备阶梯

### 入门（手机/入门相机）
| 设备 | 价格 |
|------|------|
| 手机三轴稳定器 | ¥300-800 |
| 手机滑轨 40cm | ¥100-300 |
| 八爪鱼三脚架 | ¥50-150 |

### 进阶（微单/单反）
| 设备 | 价格 |
|------|------|
| DJI RS 系列稳定器 | ¥2000-5000 |
| 滑轨 80-120cm | ¥500-2000 |
| 液压云台 | ¥500-1500 |

### 专业级
斯坦尼康 / 摇臂 / Dolly 轨道车 / Motion Control

---

## 无设备也能拍

| 需求 | 替代方案 |
|------|---------|
| 滑轨 | 坐办公椅让别人推 / 购物车 |
| 摇臂 | 自拍杆伸长缓慢升降 |
| 稳定器 | 三角支撑握法 + 小碎步 |
| 跟焦 | 手动匀速转动对焦环 |
