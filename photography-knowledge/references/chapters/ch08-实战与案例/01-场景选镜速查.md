# ch08-01 场景选镜速查

> 不知道拍什么？直接查这张表。

---

## 按场景选运镜

| 场景 | 推荐运镜 |
|------|---------|
| **美食展示** | 环绕推进 + 慢推特写 |
| **街拍/旅行** | 跟拍 + 摇镜 + 移镜 |
| **人像/情绪** | 慢推 + 环绕 + 拉 |
| **自然风光** | 升 + 摇全景 + 拉 |
| **运动/动作** | 侧跟 + 快摇 + 手持 |
| **悬疑/恐怖** | 希区柯克变焦 + 手持晃动 + 荷兰角 |
| **产品开箱** | 推 + 环绕 + 慢拉 |
| **婚礼** | 跟 + 升 + 环绕 + 斯坦尼康 |
| **Vlog 开场** | 推入环境→人物 |
| **Vlog 收尾** | 拉远人物→远景 |

---

## 按情绪选光影

| 情绪 | 光线配方 |
|------|---------|
| 浪漫 | 柔光 + 暖色温 + 逆光 + 浅景深 |
| 悬疑 | 单硬光 + 深阴影 + 青蓝色调 |
| 史诗 | 深景深 + 黄金时刻 + 低饱和 |
| 恐怖 | 底光 + 闪烁 + 极端阴影 |
| 纪实 | 自然光 + 手持 + 中性色 |

---

## AI 视频提示词模板

```
"[运镜描述] + [光线描述] + [色彩描述] + [画质描述]"

例：
"Slow orbit push-in, warm golden hour light,
 teal-orange color grade, shallow depth of field,
 cinematic, 4K"
```

**注意**：单次提示词不超过 3 个运镜指令。
