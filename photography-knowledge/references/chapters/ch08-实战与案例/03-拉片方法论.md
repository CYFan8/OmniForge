# ch08-03 拉片方法论

> 学电影最好的方式不是看书，是反复拉片。

---

## 四遍拉片法

```
第一遍：正常观看 → 记录直觉感受（哪里感动、哪里紧张）
第二遍：关声音看 → 只看构图、运镜、光影、色彩
第三遍：只听不看 → 对白节奏、音乐、音效、静默
第四遍：逐镜头分析 → 每个镜头的起幅/落幅/动机/时长
```
