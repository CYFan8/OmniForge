## 五官刻画技法

### 眼睛 —— 传神的核心

眼睛是面部最重要最复杂的结构。画好眼睛 = 画好人物的一半。

**结构层次（从内到外）**：
1. 眼眶（骨性结构，决定眼型）
2. 眼球（球体，大部分被眼睑覆盖）
3. 虹膜与瞳孔（虹膜颜色决定眼睛颜色，瞳孔永远纯黑）
4. 上眼睑（有厚度，覆盖虹膜上部约 1/5-1/4）
5. 下眼睑（薄，紧贴眼球）
6. 睫毛（上睫毛长而卷翘，下睫毛短而稀疏）
7. 泪阜（内眼角的粉色小肉球）
8. 高光点（虹膜 + 眼球表面各一个高光，位置一致）

**关键技法**：
- 眼球是球体，不是平面 → 眼白有阴影（上深下浅），纯白色是错误的
- 上眼睑投下阴影到眼球上部 → 让眼睛立体
- 双眼的视线方向必须一致（高光点在同一侧）
- 内眼角低于外眼角约 5-10 度，不是水平线

### 鼻子 —— 体积感的关键

鼻子是面部最突出的结构，但画法上「少即是多」。

**结构**：鼻根 → 鼻梁 → 鼻尖 → 鼻翼 → 鼻底 → 鼻孔

**技法要点**：
- 正面鼻子不建议勾线，用阴影暗示鼻梁两侧
- 鼻底（鼻孔下方）永远是最暗的区域
- 鼻尖高光 = 体积感的关键
- 侧面：鼻梁线从眉心开始，经过鼻梁最突出点，到鼻尖
- 男女差异：男性鼻梁更直挺，女性鼻头更小巧

### 嘴唇 —— 柔软与表情

**结构**：
- 上唇：中央唇珠（丘比特弓）+ 两侧逐渐变薄
- 下唇：中央饱满，左右各一个球状隆起
- 口裂线：上下唇的交界线，不是直线，有弧度

**技法要点**：
- 上下唇体积比例约 1:1.3（下唇稍厚）
- 上唇颜色深于下唇（受光角度不同）
- 嘴角微微上翘 = 面带善意（即使不笑）
- 唇纹用细碎短笔触，不要画成网格

### 耳朵 —— 容易被忽略的复杂结构

耳廓 → 耳轮 → 对耳轮 → 耳屏 → 对耳屏 → 耳垂 → 耳甲腔

关键：耳朵占头部侧面的 1/3，耳顶 = 眉高，耳底 = 鼻底高。
