## 人像光影造型

### 人像用光的五大经典范式

#### 1. 伦勃朗光（Rembrandt Lighting）
- 主光在人物侧前方约 45 度、高于眼线约 45 度
- 特征：暗侧脸颊出现倒三角光斑（「伦勃朗三角」）
- 适用：男性肖像，表达力量、深度、神秘感
- 光比：通常 4:1 到 8:1（戏剧性）

#### 2. 蝴蝶光 / 派拉蒙光（Butterfly Lighting）
- 主光在人物正上方偏前，镜头位置
- 特征：鼻下形成蝴蝶形阴影，颧骨下方有对称阴影
- 适用：女性肖像，柔美、优雅、经典好莱坞
- 光比：通常 2:1 到 3:1

#### 3. 环形光（Loop Lighting）
- 主光在人物侧方约 30-45 度，略高于眼线
- 特征：鼻子阴影在脸颊形成小环形，不触及嘴角阴影
- 适用：最通用的肖像光，几乎所有人都适合
- 光比：2:1 到 3:1

#### 4. 分割光（Split Lighting）
- 主光在人物正侧面 90 度
- 特征：面部一半亮一半暗，中轴线分割
- 适用：戏剧性强烈的人物，力量感，神秘
- 光比：8:1 或更高

#### 5. 逆光 / 轮廓光（Rim Lighting）
- 主光在人物后方
- 特征：人物轮廓被高光勾勒，面部处于阴影
- 适用：剪影、氛围感、情绪化表达
- 需要补光来保留面部细节时，光比可降至 2:1

### 光质的情绪语言

| 光质 | 特征 | 适合情绪 |
|------|------|---------|
| 硬光 | 阴影边缘锐利，高光集中 | 力量、紧张、戏剧化、男性 |
| 软光 | 阴影边缘柔和，高光扩散 | 柔美、浪漫、温和、女性 |
| 漫射光 | 无明显阴影方向性 | 清新、自然、生活感 |

### 人像光影实战速查

| 想要的感觉 | 用光方案 | 参考 |
|-----------|---------|------|
| 温柔浪漫 | 蝴蝶光 + 大柔光箱 + 暖色温 | 黄昏窗边 |
| 冷酷力量 | 分割光 + 硬光 + 冷色温 | 黑色电影 |
| 自然清新 | 环形光 + 窗户自然光 + 反光板补光 | 日常肖像 |
| 神秘情绪 | 逆光 + 轮廓光 + 微弱补光 | 黄金时刻轮廓 |
| 高级时尚 | 蝶形光 + 下方补光 + 高光比 | 杂志封面 |

### 面部补光技巧

- **反光板位置**：下巴下方（消除眼袋/法令纹阴影），侧面（柔化主光阴影）
- **眼神光**：光源在角膜上的反射，一个光点 = 一个光源。眼神光让眼睛「活」起来
- **避免**：顶光（眼窝黑洞）、底光（恐怖片效果）、双侧等亮光（扁平无立体）
