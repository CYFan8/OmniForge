# ch04-01 调度三层结构

> 场面调度 (Mise-en-scène) = 镜头前一切元素的组织。每个运动都有叙事目的。

---

## 三层调度

```
场面调度
├── 演员调度：站位、视线、肢体、走位
├── 镜头调度：景别、运动、角度、焦点
└── 场景调度：道具、灯光、空间、色彩
```

---

## 演员站位 = 权力关系

| 站位 | 暗示 |
|------|------|
| 面向镜头 | 开放、坦诚、挑战 |
| 背对镜头 | 逃避、拒绝、隐藏 |
| 侧身 | 防御、犹豫 |
| 画面中心 | 权力、控制 |
| 画面边缘 | 弱势、被排斥 |
| 两人极近 | 亲密或对抗 |
| 两人极远 | 疏远、隔阂 |

---

## 《婚姻故事》的调度密码

夫妻二人同框时——
- 总被门框/家具隔开
- 物理距离很近，画面却在说"他们不在一起"
- 两个近景 + 一个中景，几十秒无台词
- "表面很近，实则疏远，却剪不断"
