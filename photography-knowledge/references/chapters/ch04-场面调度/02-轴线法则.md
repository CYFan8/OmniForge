# ch04-02 轴线法则

> 180 度线是观众空间感的锚。越轴 = 故意让观众"迷路"。

---

## 180 度法则

```
          [轴线]
      A ←——————→ B
           ↓
     摄影机只能在轴线同一侧
```

越过这条线 → 观众产生空间错乱 → A 和 B 左右互换。

---

## 何时可以越轴？

| 方式 | 原理 |
|------|------|
| **长镜头交代空间** | 先让观众建立完整空间认知 |
| **参考点定位** | 用固定参照物做空间锚 |
| **运动镜头越轴** | 摄影机运动过程展示越轴全程 |
| **人物视线建新轴** | 角色看向画外→自动形成新轴线 |

### 《十二怒汉》案例

```
开场 6 分钟长镜头 → 建立空间
1 号和 7 号 = 长桌两端参考点
8 号绕桌行走 → 摄影机 U 型跟拍 → 完整呈现越轴过程
7 号被画外音吸引看 11 号 → 新轴线自然形成
```
