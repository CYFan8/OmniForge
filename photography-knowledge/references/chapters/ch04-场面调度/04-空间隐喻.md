# ch04-04 空间隐喻

> 拍的不是房子，是阶级。拍的不是楼梯，是命运。

---

## 空间符号词典

| 空间 | 隐喻 |
|------|------|
| 向上的楼梯 | 上升、希望、权力 |
| 向下的楼梯 | 坠落、绝望、地下世界 |
| 门/窗 | 机会、隔离、窥视 |
| 镜子 | 自我认知、分裂、幻觉 |
| 走廊 | 过渡、未知、压迫 |
| 地下室 | 隐藏的秘密、潜意识 |

---

## 《寄生虫》的空间三级

```
上层：富人区，阳光充足 ↗ 仰拍
中层：半地下室，昏暗逼仄 → 平视
下层：地下避难所，完全脱离社会 ↘ 俯拍

楼梯 = 阶级跨越的最核心视觉符号
大雨夜从豪宅逃回半地下 = 一路俯拍的垂直坠落
```

## 画外空间

| 方式 | 效果 |
|------|------|
| 人物视线出画 | "在看什么？"→ 悬念 |
| 声音出画 | 听到看不到 → 不安/期待 |
| 物体出画入画 | 未知力量介入 |
| 影子/倒影 | 不拍本体 → 隐藏=悬疑 |
