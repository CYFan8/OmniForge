# ch04-03 正反打

> 对话场景 90% 靠正反打。怎么"打"决定观众站在谁那边。

---

## 标准正反打

```
过肩镜头 A → 过肩镜头 B → 过肩镜头 A → ...
观众 = 对话的第三方观察者
```

---

## 变化手法

| 变化 | 效果 | 何时用 |
|------|------|--------|
| 完全正反打 | 两人平等 | 正常对话 |
| 一方只出现过肩 | 被观察、有压迫 | 审讯、权力不对等 |
| 一方特写+一方过肩 | 一方情绪更重要 | 情感爆发 |
| 正打+空镜 | 一方逃避/沉默 | 冷暴力、尴尬 |
| 跳轴正反打 | 刻意制造混乱 | 争吵、心理崩溃 |

---

## 经典运用

《沉默的羔羊》中莱克特与史达琳的对话：
- 莱克特 → 正面特写，直接看镜头
- 史达琳 → 侧面/过肩，被审视
- 权力关系用正反打的设计完成
