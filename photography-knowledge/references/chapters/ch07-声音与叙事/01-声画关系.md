# ch07-01 声画关系

> 声音占电影的 50%。关掉声音看一部"好电影"，你会发现它突然不怎么样了。

---

## 声画关系三型

| 关系 | 说明 | 例子 |
|------|------|------|
| **声画同步** | 看到什么听到什么 | 正常叙事 |
| **声画对位** | 声音和画面表达不同情感 | 欢快音乐+悲伤画面=讽刺 |
| **声画分离** | 声音来自画外/另一个时空 | 旁白、回忆、电话 |

### 经典声画对位
《发条橙》：极端暴力场景配《雨中曲》→ 暴力变得诡异而荒诞

---

## 声音的叙事功能

| 元素 | 功能 |
|------|------|
| **对白** | 推进情节、塑造人物 |
| **旁白** | 叙述、交代信息（可靠/不可靠） |
| **独白** | 揭示内心世界 |
| **音乐** | 渲染气氛、暗示主题 |
| **音效** | 增强真实感、营造空间 |
| **静默** | 最强音——制造紧张、强调冲击 |

---

## J cut 和 L cut

```
J cut：下一个场景的声音先入，画面还在上一个场景
L cut：上一个场景的声音延续，画面已切到下一个场景
```

这是 90% 流畅转场的秘密。
