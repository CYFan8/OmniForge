# ch03-03 影调风格

> 高调明亮，低调深沉。影调选择 = 全片气质。

---

## 三大影调

| 影调 | 特征 | 情绪 | 代表 |
|------|------|------|------|
| **高调 (High Key)** | 明亮、低对比 | 轻快、喜剧 | 情景喜剧 |
| **低调 (Low Key)** | 黑暗、高对比 | 神秘、悬疑 | 《教父》《七宗罪》 |
| **自然调** | 模拟自然光 | 真实、纪实 | 纪录片 |

---

## 实战配方

### 浪漫爱情
```
柔光 + 暖色温 + 逆光轮廓 + 浅景深 = 梦幻柔美
```

### 悬疑黑色
```
单光源硬光 + 深黑阴影 + 青蓝调 + 高对比 = 不安
```

### 恐怖惊悚
```
底光 + 闪烁光 + 极端阴影 + 病态绿 = 恐惧
```

### 史诗宏大
```
深景深 + 自然光 + 黄金时刻 + 低饱和 = 凝重
```
