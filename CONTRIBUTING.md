# 贡献指南

## 如何贡献一个新 Skill

1. **Fork** 本仓库
2. 在根目录创建新目录: `skill-你的技能名/`
3. 在目录内创建 `SKILL.md`，格式参考 `skill-system-core/SKILL.md`：

```markdown
---
name: skill-你的技能名
description: >-
  一句话描述这个技能做什么、何时触发。
---

# 技能名

## 1. 功能A
简短描述。

## 2. 功能B
简短描述。

## 依赖清单
| 组件 | 用途 | 安装命令 |
|------|------|---------|

## 规则联动
- 依赖的其他技能
```

4. 如果技能需要安装额外软件/插件，在「依赖清单」中写明安装方式
5. 提交 PR，在描述中说明你的技能解决了什么问题

## 质量标准

- SKILL.md 结构清晰，条目编号
- 每个功能有具体说明（不要只写功能名）
- 标注与其他技能的联动关系
- 如果经过了实机测试，在末尾加「踩坑实录」章节

## 许可

贡献的代码默认使用 MIT 许可。
