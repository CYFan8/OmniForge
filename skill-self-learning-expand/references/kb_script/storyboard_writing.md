# 短剧分镜文案

> 来源: skill-script-creation 分镜生成 + chinese_script_guide.md
> 功能: 将剧本文字转化为可供生图/建模模块使用的结构化分镜描述

## 分镜拆解规则

### 拆解粒度
- 一个镜头 = 一个场景 + 一个摄像机位 + 一组人物动作
- 场景切换 -> 新镜头
- 人物大幅位移 -> 新镜头
- 摄像机位变化 -> 新镜头
- 对话连续段：同一机位同一场景可合并为一个镜头

### 分镜字段模板
```
【镜头号】scene_01_shot_03
【场景】室内 - 咖啡馆 - 日
【人物】主角A(站位:左前), 配角B(站位:右)
【动作】A端起咖啡杯, 看向窗外; B低头看手机
【镜头】中景, 50mm, A侧45度, 焦点在A面部
【光影】窗外自然光从左侧射入, 暖色温, 轻微逆光
【氛围】安静, 略带疏离感
【台词】A: "我想了很久。" B: (沉默)
【时长】4s
```

## 镜头语言转文字规范

| 镜头类型 | 描述模板 | 焦距建议 |
|---------|---------|---------|
| 建立镜头(Establishing) | "从远处拍摄[场景]全貌, 展示环境氛围" | 24mm广角 |
| 全景(Wide) | "人物全身在画面中, 展示与环境的互动" | 35mm |
| 中景(Mid) | "人物腰部以上, 适合对话场景" | 50mm |
| 特写(Close-up) | "仅展示面部/手部/物品细节" | 85mm |
| 过肩(POV) | "从人物A肩膀后拍摄人物B" | 50mm |
| 俯拍(Top-down) | "从正上方俯视拍摄" | 35mm |

## 动作描述粒度
- 大动作: "A推门进入" -> 需要新镜头切换
- 小动作: "A微微皱眉" -> 同一镜头内, 仅标注表情变化
- 连续动作: "A起身, 走向窗户, 拉开窗帘" -> 如果摄像机跟踪, 可一个镜头; 否则拆为2-3镜头

## 台词节奏标记
- (停顿): 表示0.5-1秒的沉默
- (打断): 下一句台词不等前句说完就插入
- (重叠): 两人同时说话
- (渐弱): 台词音量逐渐降低, 通常用于场景过渡

## 分镜输出格式
- 供给生图模块: 每个镜头一式两份 -> 人物描述 + 场景描述
- 供给Blender模块: 每个镜头 -> 场景Layout(人物站位+摄像机位)
- 格式: JSON结构化, 便于MCP通道传输

## 分镜质量自检
- [ ] 相邻镜头间180度规则不违反(人物朝向一致)
- [ ] 景别变化幅度合理(不全景直接切大特写)
- [ ] 动作节奏与剧本情绪匹配
- [ ] 每镜头时长标注合理(建立镜头2-5s, 对话3-8s, 特写1-3s)
