# 剧本叙事逻辑

## 核心规则
1. 因果链完整: 每场戏有明确的因为→所以驱动力
2. 人物动机一致: 角色行为与其设定性格/目标不矛盾
3. 信息渐进: 观众获得信息的速度 = 主角获得信息的速度
4. 伏笔必收: 任何在前1/3出现的特殊物品/台词/设定，后1/3必须有回收

## 冲突密度模型
- 高冲突场 (每3句一次观点交锋): 用于高潮/转折
- 中冲突场 (每5-8句一次张力推进): 用于发展阶段
- 低冲突场 (铺垫/过渡): 用于世界观展示/人物关系建立
- 黄金比例: 高:中:低 ≈ 2:5:3

## 节奏控制
- 每15场插入一个呼吸场(低冲突，人物情感交流)
- 反转点间隔: 短剧3-5场一次，电影15-20分钟一次
- 避免连续3场以上同类型冲突(审美疲劳)

## 常见缺陷
- 人物工具化: 角色行为只为推进剧情，无自身逻辑
- 巧合过多: 超过2次巧合解决关键问题→降低可信度
- 信息倾倒: 开场大段旁白/独白解释世界观
- 情感跳跃: 角色情绪转换无过渡，上一场愤怒下一场温柔
