# 体块搭建

> 来源: skill-blender-modeling 基础建模 + 三视图导入建模

## 三视图对齐建模流程
1. 导入三视图: Front(Side=XZ) / Side(Side=YZ) / Back(Side=XZ反)
2. 设置参考透明度: 0.6, 不遮挡建模视线
3. 从最大体块开始: 躯干/主体 -> 四肢/附属 -> 细节
4. 比例锁定: 所有部分尺寸参照三视图标注

## 体块搭建六步法
1. Block Out: 最简几何体搭建整体轮廓 (Cube/Cylinder/Sphere)
2. Proportions: 调整各部分比例, 对齐三视图
3. Silhouette: 确认各角度剪影与参考图一致
4. Secondary Forms: 添加次级体块(肌肉群/机械结构)
5. Refinement: 局部细化, 增加倒角/细分
6. Check: 返回三视图逐角度比对

## 人物体块基准
- 头部: Sphere压扁 + Cylinder(脖子)
- 躯干: Cube变形为上窄下宽的梯形
- 上肢: Cylinder(上臂) + Cylinder(前臂) + 简化手掌
- 下肢: Cylinder(大腿) + Cylinder(小腿) + 简化脚掌
- 关节处: 至少2条Loop Cut保证弯曲时不变形

## 比例检查清单
- [ ] 正面三视图对齐: 总高/肩宽/腰位/膝位
- [ ] 侧面三视图对齐: 厚度/胸廓/臀部/后脑
- [ ] 背面三视图对齐: 肩胛骨位/脊柱线/发型轮廓
- [ ] 各角度剪影匹配参考图

## 常见错误
- 跳过Block Out直接做细节 -> 比例必然失调
- 不锁定比例就雕细节 -> 牵一发动全身
- 只对正面图, 侧面背面不检查 -> 厚度错误
