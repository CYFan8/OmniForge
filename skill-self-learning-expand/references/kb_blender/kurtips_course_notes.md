## 核心框架: 建模三部分

Blender建模工具分三块:
1. **左侧工具**: 选择、框选、移动/旋转/缩放等基本交互
2. **顶部工具**: 主菜单功能
3. **右侧属性**: 修改器堆栈、材质、约束等面板

关键: 看似几百个功能, 实际建模只需 <30 个功能。

## 修改器 (EP2.3核心)

### 什么是修改器
修改器 = 对基础几何体施加的**程序化、非破坏性**变换。
- 程序化: 数学计算驱动, 不手动调顶点
- 非破坏性: 不改变原始几何, 可以随时开关/删除/调整参数

### 为什么用修改器
1. 比手动建模快
2. 非破坏性: 可以随时调整回来
3. 复用: 同一个修改器参数可以被重复使用

### 关键修改器顺序 (从课程案例总结)
布尔运算(Boolean) → 重构网格(Remesh) → 细分曲面(Subdivision Surface)

### 课程中使用的修改器
| 修改器 | 用途 |
|--------|------|
| Subdivision Surface (细分曲面) | 低模变高模, 增加平滑度 |
| Shrinkwrap (收缩包裹) | 让一个物体贴合另一个表面 |
| Simple Deform (简单形变) | 弯曲/扭曲/锥化 |
| Mirror (镜像) | 对称建模 |

## 建模思维 (EP2.4核心)

### 建模马拉松方法论
1. **先保存 Ctrl+S** — Blender会崩, 养成习惯
2. **加细分修改器** — 基础cube + Subdivision Surface = 可塑形
3. **框架模式 Z → Wireframe** — 观察拓扑结构
4. **透明模式 Alt+Z** — 透过表面操作内部顶点
5. **从粗到细** — 先大形后细节

### 12个练习模型拆解
每个模型对应2-3个知识点:
- 蘑菇: Extrude + Subdivision + Shrinkwrap
- 其他模型 (重复练习)

## EP1.4案例: 戴珍珠耳环的少女

### 工作流
1. 导入参考图 (Shift+A → Image → Reference)
2. 降低不透明度 (Opacity → 0.5-0.7)
3. 中键旋转视图, 调整参考图位置
4. 从最简几何体开始搭建
5. 逐步增加细节

### 关键习惯
- 按住 Alt+Z 切换到透明模式
- 观察物体属性面板 — 选中什么, 属性就显示什么
- 参考图不遮挡建模视线即可
